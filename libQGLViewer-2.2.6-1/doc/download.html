Try the precompiled examples

The library and some of the examples are provided as binaries so that you can directly try the library.

  Get libQGLViewer-examples-linux.tar.gz (compiled on Fedora Core 4 with Qt 3). Re-compile from the sources for an optimal result is case of dependencies mismatch (see readme.txt).
  Download libQGLViewer-examples-mac.tar.gz (compiled using darwin, X11 and Qt 3).
  Download and unzip libQGLViewer-examples-win.zip (compiled using the free Qt 2.3 version with Visual Studio 6).
